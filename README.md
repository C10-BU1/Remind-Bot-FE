# Chatbot VMO

## Environment Prepare

Change env in file config/config.ts: 
- API_URL: <"Domain Backend">
- API_URL_GOOGLE_CLIENT_ID: <"Google clientId">

Install `node_modules`:

```bash
npm install
```
Start project:
```bash
npm run start
```
Start project in development enviroment:
```bash
npm run start:dev
```
